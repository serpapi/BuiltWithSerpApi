# #BuiltWithSerpApi

A community showcase of apps, agents, automations, dashboards, research tools, and experiments built with SerpApi.

This repository powers a static GitHub Pages site where builders can browse real projects, discover SerpApi engines used by each project, and find inspiration for their own search-powered ideas.

## Add Your Project

Built something with SerpApi? Open an issue and tell us about it:

[Submit a #BuiltWithSerpApi project](https://github.com/serpapi/builtwithserpapi/issues/new)

Please include:

- Project name
- Short description
- Public GitHub or GitLab link
- Hosted app link, if available
- Author or team name
- SerpApi engines used
- Tags that describe the project

We verify public links before adding them to the showcase.
